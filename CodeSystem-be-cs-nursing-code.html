BeNursingCode

Summary

Name:BeNursingCode
Status:Draft as of 2021-07-14T14:43:14+00:00
Definition:

Nursing codes in Belgium. To define a high level in nursing activities. Initially defined as supporting CodeSystem for the referral prescription.

Publisher:HL7 Belgium
Content:Fragment: A subset of the code system concepts are included in the code system resource
